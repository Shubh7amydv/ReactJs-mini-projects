# 📝 Notes App

A simple and interactive Notes Application built using pure frontend technologies. This app allows users to create, edit, and delete notes with a clean and user-friendly interface.

---

## 🚀 Features

* ✍️ Create new notes
* 🗑️ Delete notes
* ⚡ Fast and responsive UI
